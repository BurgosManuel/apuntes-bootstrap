Tablas en Bootstrap

  • Bootstrap nos proporciona unas cuantas opciones de personalización para las tablas, sin embargo, son opcionales y no son cruciales para el uso de las mismas.
  • En este caso, podemos utilizar la clase .table dentro de la etiqueta <table> para otorgarle estilos a la estructura de la tabla.
  • Una vez que agregamos la clase .table podemos sumarle a esta otras clases como .table-dark o .table-primary para elegir el color de nuestra tabla.
  • A su vez contamos con otras clases como .table-striped que le darán un efecto "rayado" a nuestra tabla.
  • Otra herramientas es la clase .table-hover que nos generará un efecto cuando pasemos el mouse por encima de nuestra tabla.
  • De esta forma podemos combinar las distintas clases y generar una tabla con el estilo que querramos, podes ver más estilos haciendo click acá para dirigirte a la documentación oficial de Bootstrap con respecto a tablas, a continuación algunos ejemplos.

Celdas o Filas de distinto color

  • Podemos otorgarle estilos individuales a cada celda o fila de la tabla.
  • Si queremos modificar una celda especifica debemos agregar la clase .table-color a los elementos <th> o <td>. Debemos modificar el "color" por aquellos que soporta bootstrap, ejemplos: .table-primary o .table-warning.
  • Si queremos modificar una fila completa debemos realizar lo mismo pero con los elementos <tr>.
